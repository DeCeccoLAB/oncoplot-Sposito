# A Tumor-Informed Peritoneal Liquid Biopsy for Risk Stratification after Curative Intent Surgery for Gastric Cancer: Results from the Prospective Liquid Study.
Sposito et al.

This repository includes:

1) Mutation_annotated_2.xlsx: Dataset of the tested variants.
2) Mutation_af.xlsx: Dataset of the identified variants for each sample.
3) script_oncoplot.R: R script to reproduce the oncoplot.
4) oncoplot.pdf: Plot of the variants detected in our cohort.

The code was run on Ubuntu 22.04.4 LTS using R (version 4.4.0).

The R packages used are:

readxl (1.4.5)
dplyr (1.2.0)
tidyr (1.3.2)
stringr (1.6.0)
maftools (2.20.0)
